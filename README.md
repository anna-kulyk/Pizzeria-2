# Pizza Delicious

Pizzeria landing page for CyberBionic Systematics FrontEnd Developer course

## Installation

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

### Production build

```bash
npm run build
```

## Technologies used

- HTML5
- CSS, SASS
- JavaScript

## Features

- Responsive Web Design
- Sections designed with Flexbox or CSS Grid
- Slider
